# Karpo Review Server
The express server for Code Review

### 사용예
1. 서버를 배포 (환경변수 GH_TOKEN, REPO_OWNER, REPO_NAME 작성)
2. github webhook 연동 (http://[YOUR_SERVER_URL]/webhook/)
